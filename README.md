### 🛡️ Shield — Child Digital Safety Platform
> *Hackathon 404 · U.S. Embassy in Mexico & StartupLab MX · April 2026*

Cross-platform parental sandbox built with **Tauri + React + Rust** in 24 hours. Features three on-device AI filtering layers (URL blocklist, multilingual NLI text classifier, MobileCLIP-S1 image classifier via ONNX Runtime) and a real-time admin dashboard with spatial heatmap — zero cloud dependency for privacy.

`Rust` `React` `Tauri` `ONNX Runtime` `NLI` `MobileCLIP`
